# Lab 8 — Formulaires HTML (Exercice 1)

Exercice de laboratoire sur les formulaires HTML avec validation et page de confirmation.

**Stack :** HTML · CSS · JavaScript Vanilla

---

## Fichiers

| Fichier | Description |
|---------|-------------|
| `lab8_exericise1.html` | Formulaire principal |
| `lab8_exercise1.js` | Validation du formulaire |
| `lab8_exercise1.css` | Styles du formulaire |
| `order.html` | Page de commande |
| `confirmation.html` | Page de confirmation |
| `confirmation.js` | Logique de confirmation |
| `confirmation.css` | Styles de confirmation |

---

## Lancer le projet

Ouvre `lab8_exericise1.html` dans un navigateur — aucune installation requise.

---

## Objectif pédagogique

- Créer et valider des formulaires HTML
- Gérer la soumission et la navigation entre pages
- Afficher une confirmation après soumission
